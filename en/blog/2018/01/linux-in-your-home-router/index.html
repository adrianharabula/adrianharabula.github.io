Linux in your home router

by Adrian Harabulă on 01 January, 2018

When you have some old routers lying around,

And default interface is not enough...

You can bring old routers back to life, by installing Linux! :D

Some things you can do:

  • #1 transform router into a reverse proxy and access it from outsite a firewalled network with ssh
  • #2 solder more RAM to your router
  • #3 solder new chip with more memory
  • #4 add usb ports to your router
  • #5 use router gpio led ports to control stuff remotely
  • #6 make you DIY router with OpenWrt and Raspberry Pi
  • and many more :D
